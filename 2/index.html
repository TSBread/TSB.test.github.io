《你好，放大器》读书笔记

by TSBread

第 1 章

  1. 首先作者说这本书是由 ADI 公司赞助的，ADI 也就是 Analog Devices（亚德诺半导体），与 TI（德州仪器）、ST（意法半导体）等均为亲华的国外半导体公司，多数产品为 DDS、ADC、DAC 等模拟芯片。

  2. 放大器的定义就是将信号和功率进行放大，以放大器为核心实现放大功能的电路称为放大电路。放大电路可以由 BJT 晶体管，俗称三极管组成。以 NPN 型三极管为例，若要进行信号放大，则条件为发射结正向偏置（P→N）、集电结反向偏置（N→P）。由此可以组成三种放大电路：共射、共基、共集放大电路，在此基础上还可以组成多级放大电路。还可以用运算放大器芯片与负反馈理论组成放大电路。

  3. 关于运放，有两个理论：虚短、虚断。所谓虚断就是运放的同相与反向输入端几乎没有电流流过，就像断路一样。虚短则是将输出端与任一相输入端连接，两点电位相同。运放的开环增益AuoA_{uo}Auo​非常大，所谓开环就是没有反馈，一味的给予输入的意思，因此信号得非常小才能在正负参考电压之内，否则只会一味的输出参考正电压或参考负电压。相对开环就有闭环，闭环增益AufA_{uf}Auf​可以通过输出电压与输入电压的比获得。以同相放大电路为例：首先，运放的同相输入电压大于反相输入电压则输出正参考电压，反之输出负参考电压，通过运放的参考手册可以得到输出端的电压曲线会在电位变化的时候有一个线性上升的过程呈短暂的线性关系，这个过程使得我们可以通过负反馈去控制输出电压。此处作者列出两个公式（1.1）：Uo=Auo⋅(Us−Ux), Ux=Uo⋅R2R2+RFU_o=A_{uo}·(U_s-U_x),\ U_x=U_o·\frac{R_2}{R_2+R_F}Uo​=Auo​⋅(Us​−Ux​), Ux​=Uo​⋅R2​+RF​R2​​，前者表明运放输出电压就是输入电压差分后乘以开环增益的值，后者就是输出电压经过电阻分压网络后得到的电压。要求三个未知量其中任意一个只要知道其余两者即可，由此可得出闭环增益关系与反相电压关系。当开环增益非常大时，闭环关系式中的 1 可忽略，便可得到闭环增益约等于电阻网络之和与反向端与 GND 之间电阻之比，反相电压中的电阻网络之和可忽略，便可得到反相电压约等于输入电压。由此一来，负反馈电路的增益仅与电阻的比例有关，与开环增益无关。

  4. 历史上第一个运放是由仙童公司的 Widlar 设计的 uA709，如今还在生产的为 1968 年诞生的 uA741。我手上也有一片该型号的运放，该运放只有一个通道，与 LM358 这类多通道运放不同的是多了两个用作偏置调零的引脚。

  5. 前面说过运放输出电压为差分输入与开环增益的乘积，而全差分运放输出不同于普通运放，其输出也为差分形式，输出端差分电压等于输入端差分与开环增益的乘积。第二个算式UOUT++UOUT−2=VOCM\frac{U_{OUT+}+U_{OUT-}}{2}=V_{OCM}2UOUT+​+UOUT−​​=VOCM​也很好理解，因为输出也为差分输出，则中心是以VOCMV_{OCM}VOCM​为参考点，就像我们一般以 GND，0V 为参考点一样。如果供电电压为 0~5V，OCM 参考电位为 2.5V，则 OUT 分别输出 5V 和 0V，OCM 也被称为共模电压。

  6. 功能放大器就是以运放为核心，将外围电路连通运放一起封装，留出输入输出供电端，就成为了功能放大器。仪表放大器也是功能放大器其中之一，具有差分对称输入，极高阻抗，常用于仪器仪表的最前端，与传感器直接相连。仪表放大器的特点是对信号的差值极其敏感，对具体共模值是多少不敏感。但它的输入管家不得悬空，也不能承载太高的电压（S 型热电偶的热电动势平均只有几毫伏，可以用该放大电路去测量）。还有差动放大器，将差分输入乘以由高精度电阻网络连接后的增益后输出、程控增益放大器，根据二进制设置的增益进行放大、压控增益放大器，其放大增益由外部施加的电压连续控制、隔离放大器，将放大器输入信号与输出信号之间的电气隔离。

  7. 放大器的选择应该从高级到低级，低级是晶体管放大器，高级是功能放大器，但是越低级的放大器成本可能会越低，针对性更强，但是设计难度和体积也会越大。一般情况下不知道要用什么放大器可以服从多数人或是在芯片厂商的手册中进行筛选。

第 2 章

  1. 输入失调电压VOSV_{OS}VOS​，英文写作 Input Offset Voltage，字面意义就是偏移电压。因为不存在理想运放，所以当两个输入端都接地时，输出端并不会输出 0。给同相输入端增加一个可调的电压，将其调整到输出为 0 时的这个电压，就被称为输入失调电压。在德州仪器的通用运放 uA741C 手册上查询得到该值最大为 6mV，典型值为 1mV（但是它有偏移调零引脚，根据手册所示连接即可手动调节到一个很优秀的值），而精密运放 OPA2205 手册上查询得到该值最大仅为 ±15uV，典型值仅为 ±2uV。作者将 1uV 以下定义为极优秀的，100uV 属于较好的，由此可见 OPA2205 属于近似极优秀的运放，同样价格也很优秀达到了两位数。如果该放大器闭环增益为 A，则当输入为 0 时也会存在一个A⋅VOSA·V_{OS}A⋅VOS​的直流电平。我手中的 uA741 就是如此，本应输出负参考电平却得到一个近似 1 伏的输出，通过芯片的调零引脚即可缓解该问题。如果是输入交流信号放大的话，可以通过耦合电容将直流分量消除。

  2. 失调电压漂移，英文写作 Input Offset Voltage Drift。定义为当温度变化、时间持续、供电电压等自变量变化时，输入失调电压会发生变化，输入失调电压随自变量变化的比值称为失调电压漂移。所以共有 3 中漂移量存在，但我在部分运放手册上只看到了以温度为自变量变化的指标，符号为dVOS/dTdV_{OS}/dTdVOS​/dT，表示指定温度范围内的平均值。对于需要高精度、高稳定性的要求，只能选择漂移系数小的放大器，该指标比失调电压更重要。

  3. 输入偏置电流IBI_BIB​，英文为 Input Bias Current。虽然前面说了运放两个输入端阻抗无穷大，相当于断路，但是并不存在理想运放，偏置电流IBI_BIB​等于两个输入端电流之和的平均值。通过查询 OPA2205 的数据手册可得该值最大为 ±0.4nA，典型值为 ±0.1nA，也就是 100000fA。作者定义优略范围为 60fA~100uA，看来该运放属于较优秀的一类。如果该指标过大，则当放大器被用作跨族放大测量外部微小电流时会分掉待测电流，或者当输入端通过一个电阻接地时，该电流会导致电阻上产生不期望的输入电压。因此为避免输入偏置电流对放大电路的影响，需要选择该指标较小的放大器，使用 FET 输入的运放会比使用 BJT 运放的指标小很多。

  4. 输入失调电流IOSI_{OS}IOS​，英文写作 Input Offset Current。定义为当输出维持在规定电平时，两个输入端流入电流的差值。该数值大小一般与其偏置电流相当，但如果偏置电流大于失调电流且为其 x 倍，则说明差值非常小。
